# Agentic coding CLIs, installed via each tool's official native installer
# (curl | sh). These ship a self-contained binary into ~/.local/bin and
# self-update, so `bootstrap.sh --agents` only does the FIRST install (it skips
# when the binary is already present) — updates are the tool's own job, the same
# way self-updating apps are excluded from winget batch upgrades.
#
# Format: <binary-to-check>|<https installer url>
# Auth (claude / codex login) is a manual first-run step and never stored here.
# In a restricted network run `proxy_on` before installing.
